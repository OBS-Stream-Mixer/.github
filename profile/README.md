# OBS Studio Software — Portable Screen Recording and Live Streaming

## About OBS Studio Software

OBS Studio Software provides professional screen recording and live streaming, widely adopted as standard by gamers, content creators, and broadcasters. OBS Studio Software combines scene composition (unlimited scenes), sources (display capture, window capture, browser source, webcam, images, text, game capture), transitions, hotkeys, filters, and audio mixer. OBS Studio Software supports streaming to Twitch, YouTube, Facebook, custom RTMP servers, and local recording to MP4, MOV, MKV, FLV. OBS Studio Software integrates noise suppression, gain control, compression, VST plugin support, and studio mode (preview before switching) directly into the interface. OBS Studio Software enables users to record tutorials, stream games, host live shows, and produce professional video content. OBS Studio Software runs on portable storage, USB drives, and local user folders. OBS Studio Software is trusted by gamers, streamers, tutorial creators, live event broadcasters, and open source advocates worldwide.
